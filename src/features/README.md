This folder is reserved for feature-scoped React modules.

Use `src/features/` when a domain grows beyond a single page and needs local:

- components
- hooks
- API adapters
- validation schemas
- feature-specific types

Recommended feature slices for future refactors:

- `loan-workflow`
- `member-import`
- `cash-control`
- `auditor`
- `dividends`
- `team-access`
